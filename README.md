# Texture Pipette

Texture Pipette is a Farming Simulator 25 mod for identifying terrain textures and paintable foliage layers.

## Features

- Detects paintable terrain textures
- Detects paintable foliage layers
- Shows matching construction menu entries where available
- Helps players and map creators identify used ground and foliage elements
