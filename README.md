# Terrain Texture And Object Picker

Terrain Texture And Object Picker is a Farming Simulator 25 mod for identifying terrain textures and supported map or construction objects.

Former name: Texture Pipette

## Features

* Detects paintable terrain textures
* Helps identify used ground textures in-game
* Supports object picking for relevant detectable map or construction objects
* Shows matching construction menu entries where available
* Helps players and map creators identify terrain and object elements during gameplay
